# VoidAddons
> Free Meteor Client addon with 58 modules for Minecraft 1.21

**[⬇ Download](https://voidaddons.netlify.app)** | **[Website](https://voidaddons.netlify.app)**

## What is VoidAddons?
VoidAddons is a free Meteor Client addon that adds 58 modules to Meteor Client. No cost, no bs — just download and play.

## Modules
Combat, Movement, Visual, Exploit and Utility modules including:
- AimAssist, AutoBedTrap, AutoPot, PacketFly, ChestAura
- FullFlight, Jetpack, BoatPhase, ExtraElytra, Glide
- SkeletonESP, NewChunks, OreSim, SoundLocator
- AutoWither, Lavacast, AutoTNT, Confuse, BlockIn
- And 39 more...

## Installation
1. Download [Meteor Client](https://meteorclient.com)
2. Download [VoidAddons.jar](https://voidaddons.netlify.app)
3. Drop `VoidAddons.jar` into `.minecraft/mods`
4. Launch Minecraft and enjoy

## Minecraft Version
`1.21.11`
